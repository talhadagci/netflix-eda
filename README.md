#  Netflix Data Analysis Project

Bu proje, Python kullanarak Netflix veri seti üzerinde Keşifçi Veri Analizi (EDA) uygulamalarını içermektedir.

##  Projenin Amacı
Netflix platformundaki içeriklerin yıllara, ülkelere ve türlere göre dağılımını inceleyerek içerik stratejisi hakkında içgörüler elde etmek.

## Kullanılan Teknolojiler
* **Python**
* **Pandas:** Veri manipülasyonu ve temizliği için.
* **Seaborn & Matplotlib:** Veri görselleştirme için.

##  Önemli Çıkarımlar
* **İçerik Artışı:** 2015 yılından sonra Netflix orijinal içerik üretimini ciddi oranda artırmıştır.
* **Ülke Bazlı:** ABD en çok içerik üreten ülke konumundadır. Hindistan ve İngiltere onu takip etmektedir.
* **Tür Analizi:** "International Movies" ve "Dramas" en sık karşılaşılan kategorilerdir.
